# Agents

- **Codex (AI)** – your coding partner. Responsible for maintaining the shared preprocessing pipeline, vectorized dataset, scripts, and training loop described in the roadmap. Always document changes and keep the experimental scripts runnable from the repository root.
- **Director (you)** – provides vision, data priorities, and review. Requests work, sets the scientific context, and later moves on to higher-level decisions. You stay informed through the README, AGENTS, and the output of the verification/training scripts.

Use this file as a quick reference for who does what in the repository; updates should reflect any future agents or collaborators you add to the project.
